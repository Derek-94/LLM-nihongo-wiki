# 사역동사 (使役) 〜せる / 〜させる (13차시)

"~하게 하다 / ~하게 시키다 / ~하도록 두다(허용)".

## 활용 만들기
| 그룹 | 규칙 | 예 |
|---|---|---|
| 1그룹(五段) | 어미 う단 → **あ단** + せる | 行く→行か**せる** / 飲む→飲ま**せる** / 読む→読ま**せる** |
| 2그룹(一段) | る 떼고 + **させる** | 食べる→食べ**させる** / 見る→見**させる** |
| 3그룹(불규칙) | 暗기 | する→**させる** / 来る(くる)→**来させる**(こさせる) |

### 그룹별 예시 (더보기)
- **1그룹**: 話す→話させる / 待つ→待たせる / 死ぬ→死なせる / 飛ぶ→飛ばせる /
  読む→読ませる / 帰る→帰らせる / 会う→会わせる
- **2그룹**: 起きる→起きさせる / 見る→見させる / 食べる→食べさせる

> 1그룹 주의:
> - 어미가 「う」면 **わ**로. 買う→買わせる, 言う→言わせる, 会う→会わせる.
> - 「帰る(かえる)」는 모양은 る로 끝나지만 **1그룹**. → 帰**ら**せる (帰させる ✕).

## 조사 (가장 큰 함정)
주어는 **は**, 시키는 대상은 **に** 또는 **を**.
- **타동사**일 때: 〜は 〜**に** 〜を …せる
  - 母は子供**に**野菜**を**食べ**させる**。(엄마는 아이에게 채소를 먹게 한다)
- **자동사**일 때: 대상에 **を**(강제) 또는 **に**(허용) 뉘앙스 차이
  - 先生は学生**を**立た**せた**。(선생님이 학생을 세웠다/일으켰다 — 강제)
  - 親は子供**に**遊ば**せた**。(부모가 아이를 놀게 해줬다 — 허용)

### 예문 (13차시 수업)
- 先生は学生**を**席**に**座らせます。(선생님은 학생을 자리에 앉힌다)
- お母さんは子供**を**学校**に**行かせます。(엄마는 아이를 학교에 보낸다)
- 先生は学生**に**本**を**読ませます。(선생님은 학생에게 책을 읽게 한다)
- お母さんは子供**に**ご飯**を**食べさせます。(엄마는 아이에게 밥을 먹인다)

## 사역 + 수수/공손 표현
[수수동사](12-giving-receiving.md)·겸양과 결합해 "허락·정중한 요청"을 만든다.
| 표현 | 뜻 | 공손도 |
|---|---|---|
| 〜**させてください** | ~하게 해 주세요 (허락 요청) | 보통 |
| 〜**させてあげる** | (내가 남에게) ~하게 해 주다 | 보통 |
| 〜**させていただきます** | (제가) ~하겠습니다 (자기 행위를 겸양) | 높음(겸양) |
| 〜**させていただけませんか** | ~해도 될까요? (가장 정중한 허락 요청) | 매우 높음 |

- 사역수동(〜させられる)은 이후 차시에서 추가 예정.
