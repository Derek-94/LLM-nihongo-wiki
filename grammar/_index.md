# 문법 지도 (Grammar Index)

차시별로 배운 문법을 모은 지도. 새 페이지가 생기면 여기에 한 줄 추가.

## 차시별
- 11차시 — [자동사·타동사 + 〜ている / 〜てある](11-자타동사.md)
- 12차시 — [수수동사 (もらう·いただく / やる·あげる·さしあげる / くれる·くださる)](12-수수동사.md)
- 13차시 — [사역동사 〜せる / 〜させる](13-사역동사.md)
- 14차시 — [수동태 〜れる / 〜られる (피해의 수동 포함)](14-수동태.md)
- 15차시 — [사역수동 〜させられる](15-사역수동.md)

## 표현·기타
- [〜によって / 〜による](n-yotte.md) — 수단·원인·기준·수동의 행위자
- [〜のに](n-noni.md) — 역접·불만 (~인데도)
- [〜ばかり](n-bakari.md) — ~만, ~만 하다 (반복·비난)
- [〜だけ / 〜しか…ない / 〜も (한정·수량)](n-shika-dake-mo.md) — 같은 양도 평가가 갈린다

## 주제 연결
- [사역](13-사역동사.md) ↔ [수수동사](12-수수동사.md): 〜させてあげる / 〜させてください
- [자동사·타동사](11-자타동사.md): 〜ている(자, 상태) vs 〜てある(타, 의도된 결과)
- [수동](14-수동태.md) ↔ [사역](13-사역동사.md) ↔ [사역수동](15-사역수동.md):
  형태 비교(〜れる/〜られる vs 〜せる/〜させる vs 〜せられる/〜させられる)
- [수동](14-수동태.md) ↔ [〜によって](n-yotte.md): 창작·발견·결정 등 수동의 행위자 표시
- [사역수동](15-사역수동.md) ↔ [〜てばかり](n-bakari.md):
  「させられてばかりいる」(시킴 당하기만 한다) 같은 결합 빈출
- [사역수동](15-사역수동.md) ↔ [〜のに](n-noni.md): 강요당한 상황에 대한 불만 표현 짝
