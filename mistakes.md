# mistakes — 틀린 것 누적

시험에서 틀린 것이 맥락과 함께 쌓인다. 다음 시험의 **1순위 출제 소스**.
형식: `날짜 · 문제 · 내 답 · 정답 · 짧은 해설`.

<!-- 예시
## 2026-05-21
- 사역 조사 — 문제: 「子供 / 野菜 / 食べる」 사역으로 / 내 답: 子供が野菜を食べさせる
  / 정답: 子供**に**野菜を食べさせる / 해설: 타동사 사역은 시키는 대상에 に.
-->

## 2026-05-25
- 단어 — 문제: 観光地(かんこうち)의 읽기와 뜻 / 내 답: 관광지 / 정답: かんこうち, 관광지 / 해설: 뜻은 맞았고 읽기까지 같이 답해야 정답 처리.
- 단어 — 문제: 商品券(しょうひんけん)의 읽기와 뜻 / 내 답: 모르겠어 / 정답: しょうひんけん, 상품권 / 해설: 상품권은 일본어로 商品券, 읽기는 しょうひんけん.
- 단어 — 문제: 伸びる(のびる)의 뜻 / 내 답: 모르겠어 / 정답: 사업이 번창하다 / 해설: 이번 캡처에서는 회사·사업 맥락으로 나온 단어라 ‘늘어나다’보다 ‘사업이 번창하다’로 기억하는 게 좋음.
