# 명사 (Nouns)

일본어 명사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 일상 / 기초
- 用事(ようじ) — 볼일, 용무
- 急用(きゅうよう) — 급한 볼일
- 部分(ぶぶん) — 부분
- 初対面(しょたいめん) — 첫 대면, 처음 만남
- 仕方(しかた) — 방법 (→ [仕方がない](adverbs-expressions.md))
- 紙(かみ) — 종이
- 例(れい) — 예
- 場合(ばあい) — 경우

## 수업 / 교실
- 席(せき) — 자리
- 自己紹介(じこしょうかい) — 자기소개
- 復習(ふくしゅう) — 복습
- 教科書(きょうかしょ) — 교과서
- 欠席(けっせき) — 결석
- 理由(りゆう) — 이유
- 質問(しつもん) — 질문
- レポート — 리포트
- 入学式(にゅうがくしき) — 입학식
- 宿題(しゅくだい) — 숙제
- 進学相談(しんがくそうだん) — 진학 상담

## 사람 / 관계
- 両親(りょうしん) — 부모님, 양친
- 相手(あいて) — 상대
- 同僚(どうりょう) — 동료
- 妻(つま) — 아내, 처
- すり — 소매치기
- 目上の人(めうえのひと) — 윗사람 (→ [수수동사](../grammar/12-수수동사.md))
- 親友(しんゆう) — 친한 친구
- 新入社員(しんにゅうしゃいん) — 신입 사원
- ガイド — 가이드, 안내인
- お母さん(おかあさん) — 엄마 (남의 어머니/내 어머니 친근형)

## 일 / 학습
- 業務(ぎょうむ) — 업무
- 教育(きょういく) — 교육
- プログラム — 프로그램
- 残業(ざんぎょう) — 잔업 (15과 [사역수동](../grammar/15-사역수동.md) 예문 단골)
- おつかい — 심부름

## 사물 / 사회
- 新聞(しんぶん) — 신문
- 外国(がいこく) — 외국
- 製品(せいひん) — 제품
- 部品(ぶひん) — 부품
- 絵(え) — 그림
- デザイン — 디자인
- デパート — 백화점
- アパート — 아파트
- 携帯電話(けいたいでんわ) — 휴대전화
- 間違い電話(まちがいでんわ) — 잘못 걸려온 전화
- デート — 데이트
- 観光地(かんこうち) — 관광지
- ファックス — 팩스
- サービス — 서비스
- カラオケ — 노래방, 가라오케
- 薬(くすり) — 약 (飲む와 짝)
- 荷物(にもつ) — 짐 (持つ와 짝)
