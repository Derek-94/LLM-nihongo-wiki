# 동사 (Verbs)

일본어 동사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 기초 동사
- 働く(はたらく) — 일하다
- やる — 하다
- 終わる(おわる) — 끝나다
- 休む(やすむ) — 쉬다
- 答える(こたえる) — 대답하다
- 座る(すわる) — 앉다
- 読む(よむ) — 읽다
- 作る(つくる) — 만들다
- 呼ぶ(よぶ) — 부르다
- 取る(とる) — 빼다, 찍다 (写真を取る 사진을 찍다)
- 死ぬ(しぬ) — 죽다
- 困る(こまる) — 곤란하다
- 疲れる(つかれる) — 피곤하다
- 笑う(わらう) — 웃다
- 捨てる(すてる) — 버리다
- 辞める(やめる) — 그만두다
- 受ける(うける) — 받다
- 受けとる(うけとる) — 수령하다, 받아 들다
- 送る(おくる) — 보내다
- 違う(ちがう) — 다르다
- 見せる(みせる) — 보여주다
- 貸す(かす) — 빌려주다
- 選ぶ(えらぶ) — 고르다, 선택하다
- 覚える(おぼえる) — 외우다, 기억하다
- 認める(みとめる) — 인정하다
- 任せる(まかせる) — 맡기다, 위임하다
- 伸びる(のびる) — 늘다, (사업이) 번창하다
- 信じる(しんじる) — 믿다
- 飲む(のむ) — 마시다 (薬を飲む 약을 먹다도 飲む)
- 持つ(もつ) — 들다, 가지다
- 泣く(なく) — 울다
- 進む(すすむ) — 나아가다, 진행되다
- 踊る(おどる) — 춤추다
- 押しつける(おしつける) — 억지로 떠맡기다 (→ 15과 [사역수동](../grammar/15-사역수동.md) 단골 행위)

## 12차시 [수수동사](../grammar/12-수수동사.md)
방향·공손도 정리는 [수수동사 페이지](../grammar/12-수수동사.md) 참고.
- やる — (아랫사람·동식물에게) 주다
- あげる — (남에게) 주다
- さしあげる — (윗사람에게) 드리다 (겸양)
- くれる — (남이 나/내 쪽에) 주다
- くださる — (윗사람이 나/내 쪽에) 주시다 (존경)
- もらう — 받다
- いただく — (윗사람에게서) 받다 (겸양)

## 감정 / 관계
- 叱る(しかる) — 야단치다, 꾸짖다
- ほめる — 칭찬하다
- 断る(ことわる) — 거절하다
- 落ち込む(おちこむ) — 낙담하다
- 期待する(きたいする) — 기대하다
- 傷つける(きずつける) — 상처 입히다
- 招待する(しょうたいする) — 초대하다
- 反対する(はんたいする) — 반대하다
- 注意する(ちゅういする) — 주의하다
- 慰める(なぐさめる) — 위로하다
- 助ける(たすける) — 돕다 (위급·곤란한 상황에서 구해 주는 결)
- 手伝う(てつだう) — 돕다 (작업·일을 거들어 주는 결)
- 教える(おしえる) — 가르치다

## 행위 / 사회
- 行う(おこなう) — 행하다, 거행하다
- 建てる(たてる) — 세우다
- 踏む(ふむ) — 밟다
- 咬む(かむ) — 물다
- 輸出する(ゆしゅつする) — 수출하다
- 輸入する(ゆにゅうする) — 수입하다
- 展示する(てんじする) — 전시하다
- 紹介する(しょうかいする) — 소개하다
- 案内する(あんないする) — 안내하다
- 参加する(さんかする) — 참가하다

## 표현 단위로 외우기
- 声に出す(こえにだす) — 소리 내다
- レポートを出す(レポートをだす) — 리포트를 제출하다
- うそをつく — 거짓말을 하다 (※ つく는 한자 안 쓰는 게 보통)
- 薬を飲む(くすりをのむ) — 약을 먹다 (먹다지만 동사는 飲む)
- 荷物を持つ(にもつをもつ) — 짐을 들다

## 14차시 [수동형](../grammar/14-수동태.md) 짝
수업에서 다룬 수동형 변환을 한 줄에 모아둠. 자세한 규칙은 [수동태](../grammar/14-수동태.md).
- 行う → 行われる
- 読む → 読まれる
- ほめる → ほめられる
- 招待する → 招待される
- 踏む → 踏まれる
- 咬む → 咬まれる
- 笑う → 笑われる
- 来る → 来(こ)られる
- 捨てる → 捨てられる
- 作る → 作られる
- 死ぬ → 死なれる
- 降る → 降られる
