# 수동태 (受動態) 〜れる / 〜られる (14차시)

"~당하다 / ~받다 / (피해의) ~를 당해서 곤란하다".

## 활용 만들기
| 그룹 | 규칙 | 예 |
|---|---|---|
| 1그룹(五段) | 어미 う단 → **あ단** + れる | 読む→読ま**れる** / 踏む→踏ま**れる** / 笑う→笑わ**れる** / 降る→降ら**れる** |
| 2그룹(一段) | る 떼고 + **られる** | 食べる→食べ**られる** / ほめる→ほめ**られる** / 捨てる→捨て**られる** |
| 3그룹(불규칙) | 暗기 | する→**される** / 来る(くる)→**来られる**(こられる) |

> 1그룹 주의:
> - 어미가 「う」면 **わ**로. 笑う→笑**わ**れる, 言う→言**わ**れる, 買う→買**わ**れる.
> - 「死ぬ」도 1그룹. → 死**な**れる.
> - 사역(〜せる/〜させる)과 모양이 비슷하니 [사역](13-사역동사.md)과 비교해서 외울 것.

### 그룹별 예시 (더보기)
- **1그룹**: 行う→行われる / 踏む→踏まれる / 咬む→咬まれる / 笑う→笑われる / 作る→作られる / 死ぬ→死なれる / 降る→降られる / 読む→読まれる
- **2그룹**: ほめる→ほめられる / 捨てる→捨てられる / 建てる→建てられる
- **3그룹**: 招待する→招待される / 来る→来られる(こられる)

## 수동형의 쓰임

수동형은 크게 **일반 수동**과 **피해의 수동** 두 갈래로 나뉜다.

### 1) 일반적인 수동 (사실 서술)
"A는 B에게 ~당하다/받다". 행위 받는 쪽이 주어.
- A は B に Vれる
- 능동문 「B が A を 〜する」 → 수동문 「A は B に 〜される」 로 변환 가능 (행위자가 있으면 に).

#### 예문
- 入学式は明日の午前9時に**行われます**。(입학식은 내일 오전 9시에 거행됩니다)
- この新聞は韓国で一番多く**読まれています**。(이 신문은 한국에서 가장 많이 읽히고 있습니다)
- 先生に**ほめられました**。(선생님에게 칭찬받았습니다)
- 友達からパーティーに**招待されました**。(친구에게 파티에 초대받았습니다)
- ビールはむぎから**作られている**。(맥주는 보리로 만들어진다 — 무생물 주어, 행위자 생략)

### 2) 피해의 수동 (迷惑の受身)
"내가 손해/곤란을 입었다"는 뉘앙스가 핵심. 일반 수동과 달리 **자동사도 가능**.
- 사람·자연 등 외부 사건이 일어나서 **내가 영향을 받음**.
- 한국어로는 능동형으로 번역되는 경우가 많아 직관적으로 어렵다.

#### 예문
- 急に雨に**降られて**困っています。(갑자기 비를 맞아서 곤란합니다 — 자동사 降る의 피해 수동)
- 私は友達に**来られて**仕事ができませんでした。(친구가 와 버려서 일을 못 했습니다)
- まわりの人に**笑われて**恥ずかしかったです。(주위 사람들에게 비웃음당해서 창피했습니다)
- 夜中、間違い電話を**かけられて**、寝られなかった。(한밤중에 잘못 걸린 전화를 받아서 못 잤다)
- 急に同僚に会社を**辞められて**困っている。(갑자기 동료가 회사를 그만둬 버려서 곤란하다)
- 私は父に**死なれた**。(아버지가 돌아가셔서 (내가) 타격을 받았다)
- 私は母にマンガを**捨てられた**。(엄마가 내 만화를 버려 버렸다 — 소유물 피해)
- 電車の中で足を**踏まれました**。(전철에서 발을 밟혔습니다 — 신체 부위 피해)
- 犬に手を**咬まれました**。(개에게 손을 물렸습니다)

> 신체·소유물 피해 패턴: **〜は 〜に 〜を Vれる** (소유물/부위는 を, 가해자는 に).

## 조사 정리
| 관계 | 조사 | 예 |
|---|---|---|
| 피해/행위를 받는 쪽(주어) | は / が | 私**は**先生に叱られた |
| 행위자(가해자) | に | 先生**に**叱られた |
| 출처가 사람이 아닐 때 (정보·초대 등) | から도 가능 | 友達**から**招待された |
| 신체/소유물 등 영향 받는 대상 | を | 足**を**踏まれた |
| 창작·발견·결정 등의 행위자 | によって | 『源氏物語』は紫式部**によって**書かれた → [〜によって](n-yotte.md) |

## 능동 ↔ 수동 변환 감각
- 능동: 先生**が**私**を**叱った。(선생님이 나를 야단쳤다)
- 수동: 私**は**先生**に**叱**られた**。(나는 선생님에게 야단맞았다)
- 수동문에 「Aがした」 형태로 행위자가 보이면 능동으로 복원 가능 — 거꾸로 검산할 때 유용.

## 헷갈리는 짝: 수동 vs 사역 vs 가능
모두 어미가 비슷하므로 형태와 의미를 같이 익힌다.
| 형태 | 의미 | 예 |
|---|---|---|
| 〜れる/〜られる (수동) | 당하다 | 叱**られる** (혼나다) |
| 〜せる/〜させる ([사역](13-사역동사.md)) | 시키다/허용하다 | 叱**らせる** (혼내게 시키다) |
| 〜られる (2그룹 가능형) | 할 수 있다 | 食べ**られる** (먹을 수 있다 / 먹힌다) — 문맥 의존 |
| 〜させられる ([사역수동](15-사역수동.md)) | 억지로 ~하게 됨 | 叱**られる**(혼나다) → 叱**らせられる**(혼나게 시킴 당하다)는 잘 안 씀. 일반적인 사역수동 예: 飲ま**せられる**(억지로 마시다) |
