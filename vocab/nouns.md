# 명사 (Nouns)

일본어 명사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 일상 / 기초
- 用事(ようじ) — 볼일, 용무
- 急用(きゅうよう) — 급한 볼일
- 部分(ぶぶん) — 부분
- 初対面(しょたいめん) — 첫 대면, 처음 만남
- 仕方(しかた) — 방법 (→ [仕方がない](adverbs-expressions.md))
- 紙(かみ) — 종이

## 수업 / 교실
- 席(せき) — 자리
- 自己紹介(じこしょうかい) — 자기소개
- 復習(ふくしゅう) — 복습
- 教科書(きょうかしょ) — 교과서
- 欠席(けっせき) — 결석
- 理由(りゆう) — 이유
- 質問(しつもん) — 질문
- レポート — 리포트
- 入学式(にゅうがくしき) — 입학식

## 사람 / 관계
- 両親(りょうしん) — 부모님, 양친
- 相手(あいて) — 상대
- 同僚(どうりょう) — 동료
- 妻(つま) — 아내, 처
- すり — 소매치기

## 사물 / 사회
- 新聞(しんぶん) — 신문
- 外国(がいこく) — 외국
- 製品(せいひん) — 제품
- 部品(ぶひん) — 부품
- 絵(え) — 그림
- デザイン — 디자인
- デパート — 백화점
- アパート — 아파트
- 携帯電話(けいたいでんわ) — 휴대전화
- 間違い電話(まちがいでんわ) — 잘못 걸려온 전화
- デート — 데이트

## 시간 / 상태
- 夜中(よなか) — 한밤중
- 小言(こごと) — 잔소리
- 利用(りよう) — 이용
- 禁止(きんし) — 금지
