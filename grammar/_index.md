# 문법 지도 (Grammar Index)

차시별로 배운 문법을 모은 지도. 새 페이지가 생기면 여기에 한 줄 추가.

## 차시별
- 11차시 — [자동사·타동사 + 〜ている / 〜てある](transitivity.md)
- 12차시 — [수수동사 (もらう·いただく / やる·あげる·さしあげる / くれる·くださる)](giving-receiving.md)
- 13차시 — [사역동사 〜せる / 〜させる](causative.md)

## 주제 연결
- [사역](causative.md) ↔ [수수동사](giving-receiving.md): 〜させてあげる / 〜させてください
- [자동사·타동사](transitivity.md): 〜ている(자, 상태) vs 〜てある(타, 의도된 결과)
