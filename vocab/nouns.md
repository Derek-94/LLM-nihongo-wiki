# 명사 (Nouns)

일본어 명사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 일상 / 기초
- 用事(ようじ) — 볼일, 용무
- 急用(きゅうよう) — 급한 볼일
- 部分(ぶぶん) — 부분
- 初対面(しょたいめん) — 첫 대면, 처음 만남

## 수업 / 교실
- 席(せき) — 자리
- 自己紹介(じこしょうかい) — 자기소개
- 復習(ふくしゅう) — 복습
- 教科書(きょうかしょ) — 교과서
- 欠席(けっせき) — 결석
- 理由(りゆう) — 이유
- 質問(しつもん) — 질문
- レポート — 리포트
