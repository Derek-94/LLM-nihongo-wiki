# 수수동사 (授受動詞) (12차시)

주고받음을 나타내는 동사. 헷갈리는 이유 = **방향 × 공손도** 2축이기 때문.

## 한눈에 보기
| 동사 | 방향 | 공손도 | 비고 |
|---|---|---|---|
| やる | 내가 → 남(아랫사람·동식물) | 낮음 | 요즘은 あげる가 더 흔함 |
| あげる | 내가 → 남 | 보통 | |
| さしあげる | 내가 → 윗사람 | 높음(겸양) | |
| くれる | 남 → 나/내 쪽 | 보통 | 주어가 상대방 |
| くださる | 윗사람 → 나/내 쪽 | 높음(존경) | |
| もらう | 내가 받음 | 보통 | 주는 사람은 から/に |
| いただく | 윗사람에게서 받음 | 높음(겸양) | |

## 핵심 감각
- **あげる계열**: 내가/제3자가 *남에게* 준다. (시점이 주는 쪽)
- **くれる계열**: 남이 *나(내 쪽)에게* 준다. → 나에게 오는 것만 くれる/くださる.
- **もらう계열**: 내가 *받는다*. 받는 쪽이 주어.

### 예문 (사물 주고받기)
- 私は友達にプレゼントをあげた。(나는 친구에게 선물을 줬다)
- 友達が私にプレゼントをくれた。(친구가 나에게 선물을 줬다)
- 私は友達にプレゼントをもらった。(나는 친구에게 선물을 받았다)
- 先生にお花を**さしあげます**。(선생님께 꽃을 드립니다 — 겸양)
- 母は私に小遣いを**くれました**。(엄마가 나에게 용돈을 줬다)
- 彼は私にコンサートのチケットを**くれました**。(그가 나에게 콘서트 티켓을 줬다)
- 先生は弟に本を**くださいました**。(선생님이 동생에게 책을 주셨다 — 존경)

> 주의: さしあげる는 본래 **윗사람에게** 드리는 표현. 교재에 「友達にさしあげる」 같은
> 예문이 보이면 활용 연습용일 뿐, 실제 회화에서 친구 상대로는 보통 あげる를 쓴다.

## 〜て + 수수동사 (행위의 주고받음)
동사 て형 + 수수동사 → "~해 주다/받다".
방향과 공손도는 사물 수수와 동일하게 작동한다.

| 표현 | 방향 | 공손도 |
|---|---|---|
| 〜て**あげる** | 내가/제3자가 남에게 (행위를) 해 주다 | 보통 |
| 〜て**くれる** | 남이 나/내 쪽에 해 주다 | 보통 |
| 〜て**くださる** | 윗사람이 나/내 쪽에 해 주시다 | 높음(존경) |
| 〜て**もらう** | 내가 남에게서 받다 / 남에게 ~해 받다 | 보통 |
| 〜て**いただく** | 윗사람에게서 받다 (겸양) | 높음(겸양) |

### 예문 (행위의 주고받음)
- 大変なとき、友達にたくさん助け**てもらいました**。(힘들 때 친구한테 많이 도움받았다)
- ガイドに観光地を案内し**てもらいました**。(가이드가 관광지를 안내해 줬다)
- 先生に日本の歌を教え**ていただきました**。(선생님이 일본 노래를 가르쳐 주셨다 — 겸양)
- 先生が本を買っ**てくださいました**。(선생님이 책을 사 주셨다 — 존경)
- 私は兄に本を買っ**てあげました**。(나는 형에게 책을 사 줬다)
- 私は妹に料理を作っ**てあげました**。(나는 여동생에게 요리를 만들어 줬다)

`〜させてください`(~하게 해 주세요)로 확장됨.
