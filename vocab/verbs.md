# 동사 (Verbs)

일본어 동사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 기초 동사
- 働く(はたらく) — 일하다
- やる — 하다
- 終わる(おわる) — 끝나다
- 休む(やすむ) — 쉬다
- 答える(こたえる) — 대답하다
- 座る(すわる) — 앉다
- 読む(よむ) — 읽다
- 作る(つくる) — 만들다
- 呼ぶ(よぶ) — 부르다
- 取る(とる) — 빼다, 찍다 (写真を取る 사진을 찍다)
- 死ぬ(しぬ) — 죽다
- 困る(こまる) — 곤란하다
- 疲れる(つかれる) — 피곤하다
- 笑う(わらう) — 웃다
- 捨てる(すてる) — 버리다
- 辞める(やめる) — 그만두다

## 감정 / 관계
- 叱る(しかる) — 야단치다, 꾸짖다
- ほめる — 칭찬하다
- 断る(ことわる) — 거절하다
- 落ち込む(おちこむ) — 낙담하다
- 期待する(きたいする) — 기대하다
- 傷つける(きずつける) — 상처 입히다
- 招待する(しょうたいする) — 초대하다
- 反対する(はんたいする) — 반대하다
- 注意する(ちゅういする) — 주의하다

## 행위 / 사회
- 行う(おこなう) — 행하다, 거행하다
- 建てる(たてる) — 세우다
- 踏む(ふむ) — 밟다
- 咬む(かむ) — 물다
- 輸出する(ゆしゅつする) — 수출하다
- 輸入する(ゆにゅうする) — 수입하다
- 展示する(てんじする) — 전시하다

## 표현 단위로 외우기
- 声に出す(こえにだす) — 소리 내다
- レポートを出す(レポートをだす) — 리포트를 제출하다

## 14차시 [수동형](../grammar/passive.md) 짝
수업에서 다룬 수동형 변환을 한 줄에 모아둠. 자세한 규칙은 [수동태](../grammar/passive.md).
- 行う → 行われる
- 読む → 読まれる
- ほめる → ほめられる
- 招待する → 招待される
- 踏む → 踏まれる
- 咬む → 咬まれる
- 笑う → 笑われる
- 来る → 来(こ)られる
- 捨てる → 捨てられる
- 作る → 作られる
- 死ぬ → 死なれる
- 降る → 降られる
