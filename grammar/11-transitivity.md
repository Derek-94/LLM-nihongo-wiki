# 자동사·타동사 + 〜ている / 〜てある (11차시)

## 자동사(自動詞) vs 타동사(他動詞)
- **자동사**: 주어가 스스로 그렇게 됨. 조사 **が**. 例) ドア**が**開く(あく)
- **타동사**: 누군가가 대상에 작용함. 조사 **を**. 例) ドア**を**開ける(あける)

### 자주 나오는 짝 (자/타 페어)
| 자동사 (が) | 타동사 (を) | 뜻 |
|---|---|---|
| 開く(あく) | 開ける(あける) | 열리다 / 열다 |
| 閉まる(しまる) | 閉める(しめる) | 닫히다 / 닫다 |
| つく | つける | 켜지다 / 켜다 |
| 入る(はいる) | 入れる(いれる) | 들어가다 / 넣다 |
| 出る(でる) | 出す(だす) | 나오다 / 내다 |
| 始まる(はじまる) | 始める(はじめる) | 시작되다 / 시작하다 |

## 〜ている vs 〜てある (상태 표현)
함정 포인트. 둘 다 "~된 상태"지만 뉘앙스가 다르다.

| 형태 | 동사 | 의미 | 예문 |
|---|---|---|---|
| 자동사 + **ている** | 自 | 자연스러운 **결과 상태** | ドアが開い**ている** (문이 열려 있다) |
| 타동사 + **てある** | 他 | **누군가 의도적으로** 해 둔 상태 | ドアが開け**てある** (문을 열어 두었다) |

- `〜てある`는 "누가 일부러 해놨다"는 의도가 깔린다. 대상은 보통 **が**를 받음: 窓が開けてある。
- `〜ている`는 주체의 의도와 무관한 현재 상태.
