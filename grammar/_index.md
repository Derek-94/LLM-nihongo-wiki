# 문법 지도 (Grammar Index)

차시별로 배운 문법을 모은 지도. 새 페이지가 생기면 여기에 한 줄 추가.

## 차시별
- 11차시 — [자동사·타동사 + 〜ている / 〜てある](11-transitivity.md)
- 12차시 — [수수동사 (もらう·いただく / やる·あげる·さしあげる / くれる·くださる)](12-giving-receiving.md)
- 13차시 — [사역동사 〜せる / 〜させる](13-causative.md)
- 14차시 — [수동태 〜れる / 〜られる (피해의 수동 포함)](14-passive.md)

## 주제 연결
- [사역](13-causative.md) ↔ [수수동사](12-giving-receiving.md): 〜させてあげる / 〜させてください
- [자동사·타동사](11-transitivity.md): 〜ている(자, 상태) vs 〜てある(타, 의도된 결과)
- [수동](14-passive.md) ↔ [사역](13-causative.md): 형태 비교(〜れる/〜られる vs 〜せる/〜させる),
  사역수동(〜させられる)은 두 개의 결합
