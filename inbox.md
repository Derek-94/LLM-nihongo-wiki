## 2026-05-21 23:18
이거 배웠어. 13차시 수업이였고, 사역 동사.

아래 하단에 단어들 보여? 자리, 앉다, 하다, 자기소개, .. 이런것들도 단어장에 적머줘

첨부 이미지: 13차시 사역 표현 교재 사진
- 주제: 사역 표현 / ~させる
- 예문: 先生は学生を席に座らせます。/ お母さんは子供を学校に行かせます。/ 先生は学生に本を読ませます。/ お母さんは子供にご飯を食べさせます。
- 관련 표현: ～させてください / ～させていただきます / ～させていただけませんか
- 하단 단어: 席(せき) 자리 / 座る(すわる) 앉다 / やる 하다 / 自己紹介(じこしょうかい) 자기소개 / 終わる(おわる) 끝나다 / 休む(やすむ) 쉬다

## 2026-05-21 23:26
1그룹, 2그룹, 3그룹 어떻게 변형되는지도 13차시로 배웠어.

첨부 이미지: 동사의 사역형 변형표 사진
- I그룹(5단 동사): あ단 + せる
  - 行く→行かせる / 話す→話させる / 待つ→待たせる / 死ぬ→死なせる / 飛ぶ→飛ばせる / 読む→読ませる / 帰る→帰らせる
  - 예외: 会う→会わせる
- II그룹(상하1단 동사): 어간 + させる
  - 起きる→起きさせる / 見る→見させる / 食べる→食べさせる
- III그룹(불규칙 동사)
  - 来る→来させる / する→させる

첨부 이미지: 13차시 사역 표현 교재 단어 영역 사진
- 예문: レポートを出してください。
- 단어 후보:
  - 復習(ふくしゅう) 복습
  - 少しずつ(すこしずつ) 조금씩
  - 声に出す(こえにだす) 소리 내다
  - 教科書(きょうかしょ) 교과서
  - 欠席(けっせき) 결석
  - 理由(りゆう) 이유
  - 質問(しつもん) 질문
  - 答える(こたえる) 대답하다
  - レポートを出す(レポートをだす) 리포트를 제출하다
