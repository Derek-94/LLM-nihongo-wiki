# 수수동사 (授受動詞) (12차시)

주고받음을 나타내는 동사. 헷갈리는 이유 = **방향 × 공손도** 2축이기 때문.

## 한눈에 보기
| 동사 | 방향 | 공손도 | 비고 |
|---|---|---|---|
| やる | 내가 → 남(아랫사람·동식물) | 낮음 | 요즘은 あげる가 더 흔함 |
| あげる | 내가 → 남 | 보통 | |
| さしあげる | 내가 → 윗사람 | 높음(겸양) | |
| くれる | 남 → 나/내 쪽 | 보통 | 주어가 상대방 |
| くださる | 윗사람 → 나/내 쪽 | 높음(존경) | |
| もらう | 내가 받음 | 보통 | 주는 사람은 から/に |
| いただく | 윗사람에게서 받음 | 높음(겸양) | |

## 핵심 감각
- **あげる계열**: 내가/제3자가 *남에게* 준다. (시점이 주는 쪽)
- **くれる계열**: 남이 *나(내 쪽)에게* 준다. → 나에게 오는 것만 くれる/くださる.
- **もらう계열**: 내가 *받는다*. 받는 쪽이 주어.

### 예문
- 私は友達にプレゼントをあげた。(나는 친구에게 선물을 줬다)
- 友達が私にプレゼントをくれた。(친구가 나에게 선물을 줬다)
- 私は友達にプレゼントをもらった。(나는 친구에게 선물을 받았다)
- 先生が本を貸してくださった。(선생님이 책을 빌려주셨다 — 존경)

## 〜て + 수수동사 (행위의 주고받음)
동사 て형 + 수수동사 → "~해 주다/받다". [사역](causative.md)과 결합하면
`〜させてあげる`(~하게 해 주다), `〜させてください`(~하게 해 주세요)로 확장됨.
