# 부사·표현 (Adverbs / Expressions)

부사, 부사구, 짧은 표현, 형용사를 모아두는 페이지.
복습 메타데이터는 [srs.md](../srs.md)에서 관리.

## 기초 표현
- 少しずつ(すこしずつ) — 조금씩
- 仕方(しかた)がない — 어쩔 수 없다 (어떻게 해도 안 된다)
- ～なんて — ~하다니, ~따위 (감정·놀람·경시 뉘앙스)
- この間(このあいだ) — 요전, 지난번
- いろいろな — 여러 가지의 (연체사)
- 普通(ふつう) — 보통; 일반적으로
- ずいぶん — 상당히, 꽤 (raw 캡처에서는 「すいぶん」으로 적혔으나 표준 표기는 ずいぶん)
- 自分(じぶん)から — 스스로, 제 발로 (자발성 강조)

## 감정·관용 표현
- 頭(あたま)に来(く)る — 열 받다, 화나다
- お気(き)の毒(どく)に — 안됐다, 유감이다 (상대의 불행에 공감)

## 의태어 / 의성어 (〜する)
- ごろごろする — 뒹굴거리다, 빈둥거리다 (집에서 아무것도 안 하고 굴러다닐 때)
- うろうろする — 서성거리다, 어슬렁거리다 (목적 없이 왔다 갔다)
- 혼동 주의: **ごろごろ**(누워서 굴러) vs **うろうろ**(걸어서 서성) — 발의 유무로 기억.

## 문법 연결 표현
- ～によって — ~에 의해 / ~에 따라 / ~로 인해 → [〜によって](../grammar/n-yotte.md)
- ～てくれる / ～てもらう / ～ていただく — 행위의 주고받음 → [수수동사](../grammar/12-수수동사.md)
- ～のに — ~인데도 (역접·불만) → [〜のに](../grammar/n-noni.md)
- ～ばかり / ～てばかりいる — ~만 한다 → [〜ばかり](../grammar/n-bakari.md)
- ～だけ / ～しか…ない / ～も — 한정·수량 → [〜だけ / 〜しか / 〜も](../grammar/n-shika-dake-mo.md)

## い형용사
- 恥ずかしい(はずかしい) — 부끄럽다, 창피하다
- ひどい — 심하다, 너무하다

## な형용사
- 元気(げんき) — 건강함, 기운참
- 真面目(まじめ) — 성실함, 진지함
- 大丈夫(だいじょうぶ) — 괜찮음
- 失礼(しつれい) — 실례임, 무례함 (명사·な형용사 양쪽 활용)
- いやだ — 싫다 (嫌だ로도 표기)
- 必要(ひつよう)だ — 필요하다
