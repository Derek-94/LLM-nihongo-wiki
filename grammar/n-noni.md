# 〜のに (역접·불만)

"~인데, ~인데도, ~한데도 불구하고". 기대와 다른 결과가 나왔을 때 쓰는 역접 접속.
단순 역접인 〜が/〜けど와 달리 **놀람·불만·아쉬움·비난** 같은 화자의 감정이 강하게 깔린다.

## 접속 형태
| 품사 | 형태 | 예 |
|---|---|---|
| 동사 | 보통형 + のに | 行く**のに** / 行った**のに** / 行かない**のに** |
| い형용사 | 보통형 + のに | 安い**のに** / 寒かった**のに** |
| な형용사 | 어간 + **な**のに | 静か**な**のに / 必要**な**のに |
| 명사 | 명사 + **な**のに | 学生**な**のに / 日曜日**な**のに |

> な형용사·명사 뒤에서는 「だ」가 아니라 **な**가 붙는 점이 함정.
> 「学生だのに」 ✕ → 「学生**な**のに」 ○.

## 의미와 뉘앙스
- A のに B: "A인데도 (당연히 예상되는 흐름과 달리) B다".
- 회화에서는 끝에 「〜のに。」로 말꼬리를 흐려 **원망·아쉬움**을 드러내기도 한다.
  - せっかく作った**のに**…。(애써 만들었는데…)
- 단순 정보 대비는 〜が/〜けど가 맞고, 감정이 실리면 〜のに가 자연스럽다.

## 예문
- 残業**な**のに、お金ももらえない。(잔업인데 돈도 못 받는다 — 불만)
- 必要じゃない**のに**、押しつけられた。(필요하지도 않은데 떠맡겨졌다)
- 一生懸命勉強した**のに**、試験に落ちた。(열심히 공부했는데도 시험에 떨어졌다)
- 子供**な**のに、難しい本を読んでいる。(어린데도 어려운 책을 읽고 있다 — 놀람)
- 雨が降っている**のに**、出かけた。(비가 오는데도 나갔다)

## 〜のに vs 〜けど/〜が
| 형태 | 결 | 예 |
|---|---|---|
| 〜が / 〜けど | 단순 대비, 중립 | 高い**けど**買った。(비싸지만 샀다) |
| 〜のに | 기대와 어긋남 + 감정(불만/놀람) | 高い**のに**買った。(비싼데도 (굳이) 샀다) |
