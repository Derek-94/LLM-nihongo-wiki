# 사역동사 (使役) 〜せる / 〜させる (13차시)

"~하게 하다 / ~하게 시키다 / ~하도록 두다(허용)".

## 활용 만들기
| 그룹 | 규칙 | 예 |
|---|---|---|
| 1그룹(五段) | 어미 う단 → **あ단** + せる | 行く→行か**せる** / 飲む→飲ま**せる** / 読む→読ま**せる** |
| 2그룹(一段) | る 떼고 + **させる** | 食べる→食べ**させる** / 見る→見**させる** |
| 3그룹(불규칙) | 暗기 | する→**させる** / 来る(くる)→**来させる**(こさせる) |

> 1그룹 주의: 어미가 「う」면 **わ**로. 買う→買わせる, 言う→言わせる.

## 조사 (가장 큰 함정)
주어는 **は**, 시키는 대상은 **に** 또는 **を**.
- **타동사**일 때: 〜は 〜**に** 〜を …せる
  - 母は子供**に**野菜**を**食べ**させる**。(엄마는 아이에게 채소를 먹게 한다)
- **자동사**일 때: 대상에 **を**(강제) 또는 **に**(허용) 뉘앙스 차이
  - 先生は学生**を**立た**せた**。(선생님이 학생을 세웠다/일으켰다 — 강제)
  - 親は子供**に**遊ば**せた**。(부모가 아이를 놀게 해줬다 — 허용)

## 확장
- [수수동사](giving-receiving.md)와 결합: 〜**させてあげる**(~하게 해 주다),
  〜**させてください**(~하게 해 주세요/시켜 주세요).
- 사역수동(〜させられる)은 이후 차시에서 추가 예정.
